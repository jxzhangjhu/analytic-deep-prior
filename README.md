# Analytic deep prior
Implementation of the analytic deep prior for solving ill-posed inverse problems (http://export.arxiv.org/abs/1812.03889)

## Install

This is the list of requirements

- python = 3.6
- numpy
- matplotlib
- tensorflow
- scipy
- jupyter
